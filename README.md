# Kaveu core

This has been created using the `hardhat init`.

## Installing

```bash
yarn
```

## Compile the contract

```bash
yarn compile
```

## Deploy the contract

```bash
yarn deploy
```

## Test the contract

First copy/paste the address of the contract to the test script or uncomment the deploy section `before()`.

```bash
yarn test
```
